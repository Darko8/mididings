mididings - Documentation

import mididings

Forks

Fork([units, ...])

duplicates all events, same as [...].

NoteFork([units, ...])
ControllerFork([units, ...])
PitchBendFork([units, ...])
ProgramChangeFork([units, ...])

duplicates events of one type, all others pass unchanged.

Gates

NoteGate()
ControllerGate()
PitchBendGate()
ProgramChangeGate()

filters by event type.

Filters

PortFilter(port, ...)
ChannelFilter(channel, ...)
KeyFilter(lower, upper)
KeyFilter(range)
VelocityFilter(min, max)
ControllerFilter(num)

filters by event property.

Splits

PortSplit({port: units, ...})
ChannelSplit({channel: units, ...})

splits by channel or port.

KeySplit(key, units_lower, units_upper)
KeySplit({range: units, ...})

splits by key; only note events are affected.

VelocitySplit(threshold, units_lower, units_upper)
VelocitySplit((min, max): units, ...)

splits by velocity.

Modifiers

Port(port)
Channel(channel)

changes port or channel.

Transpose(offset)

transposes all note events.

Velocity(offset)
VelocityFixed(value)

changes velocity, either by adding an offset, or by setting it to a fixed value.

VelocityGradient(note_lower, note_upper, value_lower, value_upper)
VelocityGradientFixed(note_lower, note_upper, value_lower, value_upper)

changes velocity, using a gradient from lower to upper.

ControllerRange(num, in_min, in_max, out_min, out_max)

maps controller range in to out.

ControlChange(port, channel, num, value)
ProgramChange(port, channel, program)

creates a midi event.

Misc

Pass()

does nothing.

Discard()

discards all events.

SwitchPatch(patch)

switches to another patch.

Call(function)

calls a python function.

Print()
Print(name)

prints event data.
